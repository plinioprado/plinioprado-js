Vending machine and tests

Vending machine, coded in a plain Javascript ES6 class with no interface, that does:

  • Print inventory
  • Refill inventory
  • Re-supply change
  • Dispense inventory based on payment
  • Return change as coins (eg. $0.35 is 1 quarter and 1 dime)

The required scope was to build an application using Test Driven Development, but the coding also aimed to take advantage of the ES6 resources and best practices. More info and source code at https://github.com/plinioprado/adp-vmachine.

Quiz acceptance application

Client-side quiz app based in Javascript ES5 as the application test, with:

  • Welcome Screen
  • Quiz View
  • Completion View

This project allowed to enroll the Application Development Profesional course, without needing to attend the Web Application course first.

The coding benefited from the tools and techiques used in Angular 1.6, applied to plain ES5 javascript as better suited for such a compact application. More info and source code at https://github.com/plinioprado/adp-entrance.
