Finance Projects

Financial and Business controls - CTR series

Series of public proofs of concept for applications, experimenting technologies that can be used in Finance, or Business, control and reporting. They explore concepts in modelling and coding approaches. Some were used in actual projects in the past, others are references for future use.

They can be found in public repos whose name start with "ctr" in www.https://github.com/plinioprado

There are some front-ends projects spread from older Angular 1 and 2 to more recent ones using ReactJs.

  • ctr-client-react controls a basic invoice in ReactJs
  • ctr-prefixedloans calculates a Brazilian pre fixed loan in Angular 2
  • ctr-client-angular controls Invoices and generation of account receivables in Angular 1

They access some REST API back-ends, exploring Business Logic Layers in different languages plus Data Access Layers to diferent SQL and NoSQL databases:

  • ctr-server-java in Java/Spring + a hardcoded db, it's the base for private fork with PostgreSQL
  • ctr-server-php in plain PHP + MySQL
  • ctr-server-node in NodeJs/Express + Firebase/MongDB/PostgreSQL/json
