Finance Projects

Financial and Business controls - CTR series

Series of public Proofs Of Concept experimenting with code approaches and technologies that can be used for actual professional projects.

The latest one is an application to be used as a base for future prototypes.

  • The front-end is in ctr-client-react-base a rewrite from zero of ctr-client-react for new requirements
  • The back-end is in ctr-server-node-base a rewrite from zero of ctr-server-node for new requirements

The functional requirements are:

  • Modeling that can be used as a component or microservice (through a REST api) for other prototypes.
  • Access managed by logins by Users in multiple Role levels to multiple Tenants
  • Maintain Users, Roles, Tenants, and Settings

The non functional requirements are driven by the tech chosen to experiment with:

  • Latest technologies and coding for the stack JavaScript/React/NodeJs/PostgreSQL
  • Priority in maximizing clean code and minimizing dependencies
  • One component can handle multiple auxiliary tables using an approach where the request brings data and format (with the display info). Using inheritable models in JavaScript objects (but they can be converted to OOP classes).
  • Use of Tests and MVP

There are some development notes here.

The other POCs are REST API back-ends, where the Business Logic Layers use different languages and Data Access Layers use different SQL and NoSQL databases:

  • ctr-server-java in Java/Spring + a hardcoded db, it's the base for a private fork with PostgreSQL.
  • ctr-server-php in plain PHP + MySQL.
  • ctr-server-node in NodeJs/Express + Firebase/MongDB/PostgreSQL/json.

Some are front-end projects spread from older Angular 1 and 2 to more recent ones using ReactJs.

  • ctr-client-react controls a basic invoice in ReactJs.
  • ctr-prefixedloans calculates a Brazilian pre-fixed loan in Angular 2.
  • ctr-client-angular controls Invoices and generation of account receivables in Angular 1.

The complete series can be found in public repos whose names start with "ctr" in www.https://github.com/plinioprado.
