Finance Projects

Financial and Business controls - CTR series

Series of public Proofs Of Concept experimenting with code approaches and technologies used in actual professional projects.

The latest is ctr-server-node-base, a rewriting of ctr-server-node with basically the following updates:

  • Modelling that can be used as a component or microservice for other prototypes.
  • Focus in PostgreSQL
  • Updated javascript coding, MVC approach, and unit tests

Some are REST API back-ends, exploring Business Logic Layers in different languages and Data Access Layers to different SQL and NoSQL databases:

  • ctr-server-java in Java/Spring + a hardcoded db, it's the base for a private fork with PostgreSQL
  • ctr-server-php in plain PHP + MySQL
  • ctr-server-node in NodeJs/Express + Firebase/MongDB/PostgreSQL/json

Some are front-end projects spread from older Angular 1 and 2 to more recent ones using ReactJs.

  • ctr-client-react controls a basic invoice in ReactJs
  • ctr-prefixedloans calculates a Brazilian pre-fixed loan in Angular 2
  • ctr-client-angular controls Invoices and generation of account receivables in Angular 1

The complete series can be found in public repos whose names start with "ctr" in www.https://github.com/plinioprado
